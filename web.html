Apa itu HTML?

  • Sebuah bahasa markup yang digunakan untuk membuat sebuah halaman web dan
    menampilkan berbagai informasi di dalam sebuah browser.
  • HTML berupa kode - kode tag yang menginstruksikan
    browser untuk menghasilkan tampilan sesuai dengan yang diinginkan.
  • HTML saat ini merupakan standar Internet yang didefinisikan dan
    dikendalikan penggunaannya oleh World Wide Web Consortium (W3C).

Struktur HTML :

  • Document Information
  • Document Header
  • Document Body

Dokumen HTML :

  1. Dokumen Header
    1. Page Title
      2. Judul Dari Halaman Web
    2. Scripting
      3. Tempat Client -side script yang disertakan (javascript, vbscript, jscript)
    3. Style
      4. Dipergunakan untuk mengatur bagaimana sebuah halaman web dengan beerbagai komponennya hendak ditampilkan ke dalam browser
    4. Meta
      5. Meta tags, description & keyword untuk mempermudah search engine dalam melakukan indexing

  2. Dokumen Body
    Bagian dari dokumen web yang akan ditampilkan ke user
    Bagian ini memuat :
    • Teks & Gambar
    • Link
    • Server side script (PHP, ASP, JSP)
    • Multimedia dan special programmed events (Shockwave, SWFs, Java Applets, Online Video)

-- Ciri Ciri:

  • Ekstensi file berupa .htm atau .html
  • Non case Sensitive
  • Terdiri atas tag-tag pembuka dan penutup (walaupun ada beberapa tag yang tidak memiliki penutup)
  • Tag-tag saling berpasangan dan bersarang

TAG HTML

  1. Penulisan Tag:
    • Tag Pembuka
    • Tag Penutup
    • Single Tag

  2. Tag Heading
    Berfungsi untuk menuliskan judul & sub-judul

  3. Tag Paragraf, HR, dan Br
    • Tag paragraf = Untuk membuat paragraf baru, membuat text berada dalam sebuah paragraf.
      Tag paragraf bisa memiliki Tag penutup, bisa juga tidak
    • Tag br = Break-line (Untuk Berpindah ke baris selanjutnya)
    • Tag hr = Horizontal-line (Untuk menambahkan garis horizontal)

  4. Font Tag
    Untuk mengatur penggunaan tulisan dalam halaman web (jenis tulisan, ukuran, warna, dll)
    Tag lain untuk manipulasi Font:
    • Tag b = Tulisan tebal
    • Tag i = Tulisan miring
    • Tag u = Tulisan bergaris bawah
    • Tag sub = subscript
    • Tag sup = superscript

  5. Image Tag
    Untuk memuat gambar kedalam halaman web
    Tag Image memiliki beberapa atribut antara lain :
    • src = lokasi gambar yang akan ditampilkan
    • width = ukuran lebar gambar
    • height = ukuran tinggi gambar
    • alt = deskripsi tentang gambar
    • title = judul gambar

  6. Anchor / Hyperlink Tag
    Dipergunakan untuk menghubungkan (linking) text dan image ke halaman lain
    atau bagian tertentu dari halaman yang sama dalam satu website atau website yang lain

  7. Tag Komentar
    Berfungsi sebagai pembungkus dokumen HTML atau komentar, agar tidak terbaca oleh browser
